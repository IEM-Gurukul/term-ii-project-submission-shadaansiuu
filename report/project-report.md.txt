# Greet App - Project Report

## 1. Introduction

Greet App is a simple Android application that displays a greeting message to the user.
The main purpose of this project is to demonstrate the basic concepts of Android development.

## 2. Objective

* To learn Android app development
* To understand user interface and user interaction
* To implement a simple project structure

## 3. Technology Used

* Java
* Android Studio
* Gradle

## 4. System Design

The application consists of a simple activity that displays a greeting message on the screen.

## 5. Implementation

* MainActivity was created
* Layout XML was designed
* The greeting message is displayed using a TextView

## 6. Result

The application runs successfully and displays the greeting message to the user.

## 7. Conclusion

This project helps in understanding the basic concepts of Android application development.

## 8. Future Scope

* User input functionality can be added
* Multiple greeting messages can be included
* The user interface can be further improved
